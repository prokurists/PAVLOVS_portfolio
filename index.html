Esmu elektroniskās komercijas speciālists uzņēmumā LMT Retail & Logistics. Strādāju ar uzņēmuma iekšējo CRM sistēmu un citām sistēmam. Pēc nepieciešamības piedāvāju risinājumus pasūtījuma apstrādes laika mazināšanai. Programmēšanu apgūstu pašapmācības ceļā - kursos un @CODECADEMY plaftormā. Izstrādāju lapas izmantojot HTML, CSS, BOOTSTRAP frontend valodas un PHP, MYSQL, JAVASCRIPT backend valodas.

Elektroniskās komercijas speciālists

SASNIEGUMI


Jau kopš tālā 2014. gada programmēju prieka pēc CODECADEMY platformā. Tika pabeigti dažādi kursi un iegūti vairāki sertifikāti.

Tika izveidots github profils, izveidots pirmais projekts, kuru var apskatīt aleksejs.pavlovs.lv

2020. gada 15. oktobrī ieguvu apliecību par tālākizglītību, piedaloties kursos Rīgas Tehniskajā Universitātē, programmā "WEB portālu izstrāde. Modernās tīmekļa tehnoloģijas programmatūras iztrādei."
